A fully functional e-commerce front-end application built using React.js, Redux Toolkit and Tailwind CSS with features like product listing, filters, cart system, and persistent state.
Features  - Product listing & filtering
- Add to cart / remove / quantity update
- Global state management using Redux Toolkit
- Responsive UI with Tailwind CSS
- API-based product data
